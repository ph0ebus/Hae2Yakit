# Hae2Yakit

用于转换 BurpSuite 平台中无敌的Hae信息收集插件的配置规则为yakit使用，减少因为插件不支持而无法脱离burp的烦恼

## 使用方法

### 1 安装插件

在 yakit 的插件仓库中搜索关键词 hae2yakit
